\texttt{double.ll} is a minimal C program compiled at \texttt{-O0} with full debug info (\texttt{clang -O0 -g -S -emit-llvm double.c}). It defines three functions---\texttt{@init}, \texttt{@double\_1}, and \texttt{@main}---whose control flow, heap allocation, and loop structure exercise all four LLVManim animation modes.
